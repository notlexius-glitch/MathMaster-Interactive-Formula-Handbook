Quick Revision Formulas (Exam Ready)

Algebra

(a+b)² = a² + 2ab + b²

a² - b² = (a+b)(a-b)

Quadratic: x = (-b ± √(b² - 4ac)) / 2a

Trigonometry

sin²θ + cos²θ = 1

1 + tan²θ = sec²θ

sin(α+β) = sinα cosβ + cosα sinβ

Calculus

d/dx(xⁿ) = nxⁿ⁻¹

∫xⁿdx = xⁿ⁺¹/(n+1) + C

∫eˣdx = eˣ + C

Coordinate Geometry

Distance = √[(x₂-x₁)² + (y₂-y₁)²]

Midpoint = ((x₁+x₂)/2, (y₁+y₂)/2)

Probability

P(E) = favorable / total

nCr = n! / r!(n-r)!

Statistics

Mean = Σx / n

σ = √(Σ(x-x̄)² / n)
