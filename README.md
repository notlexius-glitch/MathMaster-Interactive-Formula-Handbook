# 📚 MathMaster – Interactive Formula Handbook

## 📌 Overview
MathMaster is an interactive mathematics formula handbook designed to help students quickly revise and understand important formulas from different mathematical chapters.

It provides chapter-wise formula sheets, cheat sheets, important formula collections, and a fast search system for quick revision.

---

## 🎯 Features

### 📖 Chapter-wise Formulas
- Algebra
- Trigonometry
- Calculus
- Coordinate Geometry
- Probability
- Statistics
- Matrices
- Vectors

---

### 🔍 Search System
- Instant formula search
- Works across all visible cards
- Client-side filtering using JavaScript

---

### 📌 Important Formulas
- High priority exam formulas
- Chapter-wise revision sheet
- Quick learning format

---

### ⚡ Cheat Sheet Mode
- One-page revision format
- Only most important formulas
- Fast exam preparation tool

---

### 🎨 UI Features
- Card-based design
- Responsive layout
- Hover effects
- Clean and simple interface

---

## 🛠️ Tech Stack
