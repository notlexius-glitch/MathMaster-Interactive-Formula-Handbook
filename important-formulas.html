⭐ Important Formulas

Chapter-wise High Priority Revision Sheet

📘 Algebra

(a+b)² = a² + 2ab + b²

(a-b)² = a² - 2ab + b²

a² - b² = (a+b)(a-b)

x = (-b ± √(b² - 4ac)) / 2a

📗 Trigonometry

sin²θ + cos²θ = 1

1 + tan²θ = sec²θ

sin(α+β) = sinα cosβ + cosα sinβ

cos(α+β) = cosα cosβ - sinα sinβ

📙 Calculus

d/dx(xⁿ) = nxⁿ⁻¹

∫xⁿdx = xⁿ⁺¹/(n+1) + C

d/dx(sin x) = cos x

d/dx(cos x) = -sin x

📕 Coordinate Geometry

Distance = √[(x₂-x₁)² + (y₂-y₁)²]

Midpoint = ((x₁+x₂)/2, (y₁+y₂)/2)

Slope = (y₂ - y₁)/(x₂ - x₁)

🎲 Probability

P(E) = favorable / total outcomes

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

nCr = n! / r!(n-r)!

📊 Statistics

Mean = Σx / n

σ = √(Σ(x-x̄)² / n)

Median = middle value

🧮 Matrices

Det(2×2) = ad - bc

A⁻¹ = (1/det) × adj(A)

A × B defined if columns of A = rows of B

📐 Vectors

|a| = √(x² + y² + z²)

a · b = axbx + ayby + azbz

a × b = |a||b|sinθ
