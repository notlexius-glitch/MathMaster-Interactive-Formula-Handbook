Vector Definition

A vector is a quantity that has both magnitude and direction.

Position Vector

Vector from origin to a point (x, y, z)

Magnitude of Vector

|a| = √(x² + y² + z²)

Unit Vector

â = a / |a|

Addition of Vectors

a + b = (ax+bx, ay+by, az+bz)

Subtraction of Vectors

a - b = (ax-bx, ay-by, az-bz)

Dot Product

a · b = axbx + ayby + azbz

Dot Product Angle Form

a · b = |a||b|cosθ

Cross Product

a × b = |a||b|sinθ (direction: right-hand rule)

Scalar Triple Product

a · (b × c)

Coplanar Vectors

a · (b × c) = 0

Parallel Vectors

a = k b (scalar multiple)

Orthogonal Vectors

a · b = 0

Projection of a on b

proj = (a · b / |b|²) b
