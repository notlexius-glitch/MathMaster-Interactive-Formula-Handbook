Basic Probability

P(E) = Number of favorable outcomes / Total outcomes

Complement Rule

P(A') = 1 - P(A)

Addition Rule (General)

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Addition Rule (Mutually Exclusive)

P(A ∪ B) = P(A) + P(B)

Multiplication Rule

P(A ∩ B) = P(A) × P(B|A)

Independent Events

P(A ∩ B) = P(A) × P(B)

Conditional Probability

P(A|B) = P(A ∩ B) / P(B)

Bayes Theorem

P(A|B) = [P(B|A)P(A)] / P(B)

Permutation Formula

nPr = n! / (n - r)!

Combination Formula

nCr = n! / (r!(n - r)!)

Factorial Definition

n! = n × (n-1) × ... × 1

Probability of Impossible Event

P(∅) = 0

Probability of Sure Event

P(S) = 1
