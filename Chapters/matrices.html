Matrix Definition

A matrix is a rectangular array of numbers arranged in rows and columns.

Order of Matrix

Order = number of rows × number of columns (m × n)

Matrix Addition

Add corresponding elements of same order matrices

Matrix Subtraction

Subtract corresponding elements of same order matrices

Scalar Multiplication

Multiply each element by a constant

Matrix Multiplication

(A × B) is defined if columns of A = rows of B

Identity Matrix

I = square matrix with 1s on diagonal and 0 elsewhere

Zero Matrix

All elements are 0

Transpose of Matrix

Rows become columns (Aᵀ)

Determinant (2×2)

|A| = ad - bc

Inverse Matrix (2×2)

A⁻¹ = (1/(ad - bc)) × [[d, -b], [-c, a]]

Singular Matrix

Matrix with determinant = 0

Non-Singular Matrix

Matrix with determinant ≠ 0

Symmetric Matrix

A = Aᵀ

Skew-Symmetric Matrix

Aᵀ = -A
