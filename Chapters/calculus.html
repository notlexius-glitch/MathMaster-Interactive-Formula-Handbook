Limit Definition of Derivative

f'(x) = lim(h→0) [f(x+h) - f(x)] / h

Derivative of Constant

d/dx(c) = 0

Power Rule

d/dx(xⁿ) = nxⁿ⁻¹

Sum Rule

d/dx[f(x)+g(x)] = f'(x)+g'(x)

Difference Rule

d/dx[f(x)-g(x)] = f'(x)-g'(x)

Product Rule

d/dx[uv] = u·dv/dx + v·du/dx

Quotient Rule

d/dx(u/v) = (v·du/dx - u·dv/dx)/v²

Chain Rule

d/dx[f(g(x))] = f'(g(x))·g'(x)

Derivative of sin(x)

d/dx(sin x) = cos x

Derivative of cos(x)

d/dx(cos x) = -sin x

Derivative of tan(x)

d/dx(tan x) = sec²x

Derivative of eˣ

d/dx(eˣ) = eˣ

Derivative of ln(x)

d/dx(ln x) = 1/x

Indefinite Integral

∫f(x)dx = F(x) + C

Power Rule for Integration

∫xⁿdx = xⁿ⁺¹/(n+1) + C, n ≠ -1

Integral of 1/x

∫(1/x)dx = ln|x| + C

Integral of eˣ

∫eˣdx = eˣ + C

Integral of sin(x)

∫sin(x)dx = -cos(x) + C

Integral of cos(x)

∫cos(x)dx = sin(x) + C

Fundamental Theorem of Calculus

∫[a→b] f(x)dx = F(b) - F(a)
